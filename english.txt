NAME
       strlcpy - Copy a NUL terminated string into a sized buffer

SYNOPSIS
       size_t strlcpy(char * dest, const char * src, size_t size);

ARGUMENTS
       dest
           Where to copy the string to

       src
           Where to copy the string from

       size
           size of destination buffer
